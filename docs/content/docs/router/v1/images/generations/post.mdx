# Image Generation API

Generate images from text prompts using AI models like Google Imagen.

## Supported Models

- **imagen-4.0-generate-001**: Google Imagen 4.0 for high-quality image
  generation

## Request Parameters

- **prompt**: Text description of the image to generate
- **n**: Number of images to generate (1-8)
- **size**: Image dimensions (e.g., "1024x1024", "1920x1080")
- **quality**: "standard" or "hd" (HD costs 2x)
- **response_format**: "url" or "b64_json"

## Pricing

Images are charged per generation, not per token:

- Standard quality: $0.020 per image
- HD quality: $0.040 per image
- Platform fee: 20% added to base cost
